5. Data Storage & Retention

All data is stored within your Zoho CRM instance in custom fields. No data is stored on external servers. You can delete data at any time by clearing custom field values or uninstalling the extension.

6. Data Security

API keys are stored securely within the Sigma extension configuration. All API calls use HTTPS encryption. The extension inherits all Zoho CRM security features.
